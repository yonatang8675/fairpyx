The Problem: Restricted Max-Min Fair Allocation

Given a set of agents and indivisible items, allocate items so the minimum total value any agent receives is maximized (the "Santa Claus" problem).

This demo uses the Restricted Model:

  • Each item has a fixed size pj.
  • Each agent either wants the item (value = pj) or doesn't (value = 0).

The algorithm by Polacek & Svensson (2014) guarantees each agent gets at least OPT / (4 + ε). It uses binary search over target value T, with an alternating-tree local search at each step. [Paper]

How to Use

  1. Choose the number of agents and items, then click Build Table (or Random Input to auto-fill).
  2. Give each item a size (a positive whole number). This is the item's value.
  3. Check the box in each cell where an agent wants that item. Unchecked means the agent values it at 0.
  4. Click Run Algorithm to see the allocation on a new page.

Each agent must want at least one item. All agent and item names must be filled in and unique.

Input

Max 20 agents, 50 items, item sizes 1–1000. Check the box if an agent wants an item.

Paper Reference

Polacek, L. & Svensson, O. (2014). Quasi-Polynomial Local Search for Restricted Max-Min Fair Allocation. arXiv:1205.1373
